Usage:
To run this script:
1. Download the fortune cookie dataset
2. Extract and place the five .txt data files in the following directory:
    /data
    Make sure the /data are in the same folder as the code.
3. Run the run_code.bat, it will automatically run the code
    and produce an output.txt file that contains the outputs of the code
---------------------------------------------------
Project Structure for Decision Tree Part:
- main.py               : Main Python script that read data from the files,
                           train the data on the Naive Bayes classifier and calculate the training and test accuracies
- classifier.py         : Implementation for Naive Bayes Classifier
- preprocess.py         : Implementation for preprocessing the data
- README.txt            : Project documentation
- run_code.bat          : Executing the code and produce a output.txt file that contains the output of the program
- 11581172-Ye.pdf       : Homework solution for analytical part
